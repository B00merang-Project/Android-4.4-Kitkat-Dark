# Android-4.4-Kitkat-Dark
Dark variant of Android 4.4 Kitkat

### Manual installation

Extract the zip file to the themes directory i.e. `/home/USERNAME/.themes`

### Requirements

- GTK+ 3.20 or above
- Murrine and Pixmap theme engines
